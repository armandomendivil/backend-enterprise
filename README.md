# image-uploader-s3

## Configure environment variables

```
export host="127.0.0.1"
export db="test"
export port=27017
export username=""
export password=""
export AWS_ACCESS_KEY=""
export AWS_SECRET_KEY=""
export MAILGUN_API_KEY=""
export MAILGUN_DOMAIN=""
```

## Running Linter

```bash
npm run lint
```
